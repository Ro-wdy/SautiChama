# SautiChama - Voice Ledger for Table Banking

A modern web application for managing Table Banking (Chamas) with multi-signature approval system and voice-based notifications, designed for accessibility and financial inclusion.

### Landing Page

- **Hero Section**: Compelling introduction to SautiChama's value proposition
- **Features Overview**: Showcase of key features with icons and descriptions
- **How It Works**: Step-by-step explanation of the transaction flow
- **Problem & Solution**: Clear articulation of the problems solved
- **Responsive Design**: Mobile-first design that works on all devices
- **Green Theme**: Professional green color scheme representing trust and growth

### Admin Dashboard

- **Overview Dashboard**:
  - Real-time statistics (Total Balance, Active Members, Pending Approvals, Transactions)
  - Transaction trends chart
  - Transaction type distribution pie chart
  - Recent transactions list
- **Transactions Management**:
  - Complete transaction history
  - Search and filter functionality
  - Status tracking (Pending, Approved, Completed)
  - Multi-signature approval status
- **Members Management**:
  - Member directory with role badges
  - Contact information
  - Individual balances
  - Quick actions (Call, View details)
- **Settings**:
  - Approval configuration
  - Voice notification preferences
  - Group information management
  - API integration settings
